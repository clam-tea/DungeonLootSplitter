Logic Explanation

This website is all based around the central array, loot. "Add Loot" reads the fields and if valid, adds a { name: lootName, value: lootValue } element to loot, then calls renderLoot().

The loop is used in said renderLoot(), where a for loop steps through each element in loot and creates a <li> element with that loot's information.
